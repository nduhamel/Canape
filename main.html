canape.main – Main Canape module¶

class canape.main.Canape¶
check()¶

Process all series from database:

  1. First search for new episode with Canape.updateEpisodes()
  2. Secondly search and download them with Canape.getEpisodeDownload()
  3. Thirdly search and download subtitles with Canape.getEpisodeSubtitles()
daemon_run()¶

Launch periodically Canape.check().

The launch interval is determined by the config file:

[tvshow]
check_interval = TIME IN MINUTES
getEpisodeDownload(serieObj, episodeObj)¶

Try to retrieve a download link for a specified serie’s episode. Return an instance of the canape.video.video.Video class or None.

  • serieObj must be an instance of the canape.object.Serie class.
  • episodeObj must be an instance of the canape.object.Episode class.

Use canape.video.Searcher for find video link and canape.chooser.VideoChooser to choose the best.

getEpisodeSubtitles(serieObj, episodeObj, videoObj=None)¶

Try to retrieve subtitles for a specified serie’s episode. Return an instance of the canape.subtitle.subtitle.Subtitle class or None.

  • serieObj must be an instance of the canape.object.Serie class.
  • episodeObj must be an instance of the canape.object.Episode class.
  • videoObj must be an instance of the canape.chooser.VideoChooser

Use canape.subtitle.Searcher for find subtitle and canape.chooser.SubtitleChooser to choose the best.

updateEpisodes(serieObj)¶

Update serieObj with new avalaible episodes return updated serieObj

serieObj must be an instance of the canape.object.Serie class.

Get information by canape.information.Searcher and test airdate.
