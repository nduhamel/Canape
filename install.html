Installation¶

From source¶

1/ Clone git repository:

git clone git://github.com/nduhamel/Canape.git

2/ Install Canape:

easy_install Canape

Client¶

Canape run as a daemon, to control it you need a client. To install the default client simply clone the repository:

git clone git://github.com/nduhamel/CanapeClient.git
